# Score Formula

Final Suitability Score =

Infrastructure Readiness (30%)
+
Connectivity (25%)
+
Navigation Suitability (25%)
+
Demand & Economic Activity (20%)

Maximum Score = 100

---

# Confidence Formula

Confidence is determined using:

* Dataset Coverage
* Data Completeness
* Dataset Agreement
* Evidence Strength

Higher confidence indicates stronger agreement across supporting datasets.

# Key Showcase Message

The system does not provide unexplained scores.

For every location the dashboard can show:

* Which datasets were used
* Which records contributed
* What increased the score
* What reduced the score
* Why confidence is high or low
* Which opportunities exist
* Which constraints exist

Every intelligence output is traceable, explainable and auditable.
